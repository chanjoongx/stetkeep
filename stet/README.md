# stet

Shorthand alias for [`stetkeep`](https://github.com/chanjoongx/stetkeep).

```bash
npm install -g stet
stet install
```

`stet` is the traditional editorial mark from Latin *"let it stand"*, used by editors to cancel a proposed deletion and preserve the original text. `stetkeep` applies the same principle to code: stop Claude from "helpfully" refactoring your intentional code.

This package is a thin proxy that installs and invokes [`stetkeep`](https://www.npmjs.com/package/stetkeep) under the hood. Both `npm install stet` and `npm install stetkeep` give you the same CLI. Use whichever is shorter to type.
